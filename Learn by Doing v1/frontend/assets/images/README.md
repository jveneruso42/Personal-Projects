# Image Assets

## sad_dog_with_toy.jpg

This is the Golden Retriever photo showing a sad dog laying down with his torn toy just out of reach.

### Instructions to add the image:
1. Save the attached image as `sad_dog_with_toy.jpg` in this directory
2. The image should be the Golden Retriever photo with sunset background
3. Run `flutter pub get` to rebuild the app
4. The unauthenticated screen will now display this real photograph instead of the painted version
