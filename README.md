# Personal Projects

This repository contains various personal learning and development projects.

## Projects

### Learn by Doing v1
A simplified learning management application with authentication and dashboard functionality.

**Features:**
- Flutter Web frontend with authentication screens
- Simplified home screen with Dashboard tab
- FastAPI backend with JWT authentication  
- PostgreSQL database with User model
- Complete documentation and startup scripts

**Tech Stack:**
- Frontend: Flutter Web, Riverpod, GoRouter, Dio
- Backend: FastAPI, SQLAlchemy, PostgreSQL, Alembic
- Authentication: JWT tokens, email verification

See the project's `README.md` for detailed setup instructions.
